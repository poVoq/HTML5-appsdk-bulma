# HTML5-appsdk-bulma
Simple HTML5 app template for Ubuntu Touch made with Bulma CSS and VanillaJS.

You might want to check out the official (but outdated) [HTML5 SDK instead](https://github.com/ubports/ubuntu-html5-theme).
